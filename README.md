# Fit Reporter

**Fit Reporter** je moderní desktopová aplikace pro detailní analýzu sportovních souborů `.FIT` (ze zařízení Garmin, Wahoo, Strava, Polar a dalších) a rychlé generování přehledných tištěných reportů.

---

## 100% soukromí (Plně offline)

Aplikace funguje **zcela bez internetového připojení**. Veškerá analýza souborů, výpočty výkonnostních metrik a vykreslování tras na mapě probíhá **výhradně lokálně na vašem počítači**. Žádná vaše tréninková data (GPS souřadnice, tepové frekvence, osobní statistiky) se nikdy neodesílají na žádný externí server ani cloud.

---

## Hlavní funkce

*   **Rychlý import:** Stačí přetáhnout nebo vybrat `.FIT` soubor z vašeho tréninkového zařízení.
*   **Interaktivní přehled:**
    *   Grafy srdečního tepu, rychlosti/tempa a výškového profilu trasy.
    *   Interaktivní mapa trasy s možností přepínání stylů (Cyklistická barevná, Satelitní, Turistická).
*   **Profesionální export:**
    *   **PDF export:** Vygeneruje čistý a zalamovaný report formátu A4 připravený k tisku, bez ořezávání tabulek či grafů.
    *   **Word export (.docx):** Vytvoří plně editovatelný dokument včetně strukturovaných tabulek mezičasů, přehledů a vložených grafů.
*   **Rychlé změny vzhledu:** Výběr z barevných témat (inspirovaných Garminem, Stravou nebo minimalistický vzhled Elegant) s okamžitým přenosem do exportovaných dokumentů.
*   **Nedávné aktivity:** Rychlý přístup k naposledy analyzovaným souborům přímo z úvodní obrazovky.

## Instalace aplikace

Aplikaci nainstalujete ve dvou jednoduchých krocích:

1.  Stáhněte si nejnovější instalační soubor **`Fit_Reporter_Setup.exe`** ze sekce [Releases](https://github.com/Patrik-Rybka/fit-reporter/releases) na tomto GitHubu.
2.  Spusťte stažený soubor, potvrďte licenční podmínky a dokončete průvodce instalací.

*Poznámka: Aplikace obsahuje automatický updater. Pokud bude vydána nová verze, aplikace vás na ni při spuštění sama upozorní a nabídne stažení.*

## Jak aplikaci používat

1.  **Načtení aktivity:** Po spuštění klikněte na **Vybrat .FIT soubor** (nebo si aplikaci vyzkoušejte kliknutím na ukázkovou demo aktivitu).
2.  **Úprava reportu:** V levém panelu v záložce *Metriky* zaškrtněte, které grafy a tabulky chcete v reportu zobrazit. V záložce *Vzhled* můžete zvolit barevné téma a přidat vlastní titulek. V záložce *Poznámky* můžete připsat své pocity z tréninku.

# Fit Reporter

**Fit Reporter** is a modern desktop application for detailed analysis of `.FIT` sports files (from Garmin, Wahoo, Strava, Polar and others) and fast generation of clear printed reports.

---

## 100% privacy (Fully offline)

The application works **completely without an internet connection**. All file analysis, performance metrics calculations and route plotting on the map take place **exclusively locally on your computer**. None of your training data (GPS coordinates, heart rate, personal statistics) is ever sent to any external server or cloud.

---

## Main features

* **Quick import:** Just drag or select a `.FIT` file from your training device.
* **Interactive overview:**
* Graphs of heart rate, speed/pace and elevation profile of the route.
* Interactive route map with the ability to switch styles (Cycling color, Satellite, Hiking).
* **Professional export:**
* **PDF export:** Generates a clean and folded A4 report ready for printing, without cropping tables or graphs.
* **Word export (.docx):** Creates a fully editable document including structured split tables, overviews and embedded graphs.
* **Quick appearance changes:** Choose from color themes (inspired by Garmin, Strava or the minimalist Elegant look) with instant transfer to exported documents.
* **Recent activities:** Quick access to recently analyzed files directly from the home screen.

---

## Installing the application

You can install the application in two easy steps:

1. Download the latest installation file **`Fit_Reporter_Setup.exe`** from the [Releases](https://github.com/Patrik-Rybka/fit-reporter/releases) section on this GitHub.
2. Run the downloaded file, accept the license terms and complete the installation wizard.

*Note: The application contains an automatic updater. If a new version is released, the application will notify you of it when you start it and offer to download it.*

---

## How to use the application

1. **Loading an activity:** After starting, click **Select .FIT file** (or try the application by clicking on the sample demo activity).
2. **Editing the report:** In the left panel, in the *Metrics* tab, check which graphs and tables you want to display in the report. In the *Appearance* tab, you can choose a color theme and add your own title. In the *Notes* tab, you can add your feelings about the training.
3. **Saving:** By clicking on the buttons in the top bar (**Export PDF** / **Export Word**), select the destination folder on your computer and save the resulting file.
